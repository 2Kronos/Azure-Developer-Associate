### 1. The Potentiometer (The "Manual Control" or "Simulator")

- **Purpose:** To **manually set and hold** a specific power level for testing. It's like a dial you turn to tell the Arduino, "Assume the solar power is at _this_ level."
    
- **How it's used:** You physically turn the knob of the potentiometer. The Arduino reads this analog value (0-1023) from pin A0 and maps it to one of the four power levels (0-3).
    
- **Best for:**
    
    - **Initial testing and debugging:** You can set a fixed power level (e.g., Level 3) and make sure your washing cycle runs perfectly under ideal conditions.
        
    - **Testing specific scenarios:** You can set a fixed low power level (e.g., Level 1) and verify that the machine correctly pauses and waits.

```code
// C++ code
//
int potIn, potInVal;
void setup()
{
  pinMode(A0, INPUT);
  Serial.begin(9600);

}

void loop()
{
   potIn = analogRead(A0);
    potInVal = map(potIn, 0, 1023, 3, 0);
   Serial.println(potIn);
  	potPower(potInVal);
   delay(500);
}

void potPower(int potInVal){
  
  switch(potInVal){

      case 0:
      Serial.println("Catogoray 0: Battery charging");
      break;

      case 1:
      Serial.println("Catogoray 1: Insufficient Power");
      break;

      case 2:
      Serial.println(" Catogoray 2: Moderate Power");
      break;

      case 3:
      Serial.println("Catogoray 3: Maximum Power");
      break;


      default:
      Serial.println(" 0ff");
      break;
			
    	}	
 Serial.println("-------------------");
}
```
